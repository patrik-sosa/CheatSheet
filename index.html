HTML Cheat Sheet

brief but useful

by PSDesign

Name of the element Code representation Brief description
HTML as in "HyperText Markup Language" <HTML> HTML elemet is used to define content to a web page. Basically, it says to the web browser at the begining of the code that HTML language is to be used.
Title Element <title> As a name suggests this element contains text that represents a title of HTML documet. It is suppose to be placed it metadata section of the document. Another feature this element has is that the content of the element is displayed as a text on the web browser's tab.
Comments <!-- --> Ability to add comments in document is a very handy feature. Everything between
<!-- as an opening tag and -->
as a closing tag is being ignored and therefore not rendered as a web page by a web browser.
